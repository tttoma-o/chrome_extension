リポジトリは、プロジェクトのコードやファイルを保存する場所です。

新しいリポジトリを作成

💡 リポジトリ作成のヒント

  • リポジトリ名: プロジェクトの内容が分かりやすい名前にしましょう
  • 説明: プロジェクトの目的を簡潔に説明しましょう
  • 公開設定: 初心者の方は「Public」をお勧めします
  • README: 「Add a README file」にチェックを入れておきましょう
